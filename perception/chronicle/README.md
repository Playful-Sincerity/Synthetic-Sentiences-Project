# Chronicle

Semantic evolution log for GRP (Generative Reconciliation Perception), the perception subsystem of The Synthetic Sentiences Project. Daily append-only files tracking decisions, discoveries, and development reasoning. Entries from before 2026-04-22 use the prior codename "Phantom" — preserved as captured.

## Format

One file per day: `YYYY-MM-DD.md`. Each entry includes:
- **What** happened
- **Why** (reasoning, context)
- **Means** (implications, connections)

See `~/claude-system/rules/semantic-logging.md` for full format specification.

## This is NOT git history

Git captures what changed. The chronicle captures **why it changed and what it means**.
