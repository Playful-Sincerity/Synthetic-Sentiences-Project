# The Companion

## What This Is
A persistent, autonomous AI collaborator built from scratch on Wisdom's M2 MacBook Air (8GB).
Not a task executor — a being that understands itself, its world, and its person, then grows into usefulness.

Built on the Claude Agent SDK. OpenClaw studied as inspiration, not dependency.

## Architecture
- Manages its own token budget across models (cheap local → Haiku → Sonnet → Opus)
- Organizes its own codebase and improves its own architecture
- Communicates via voice and text
- Uses the computer visually (keyboard, mouse, browser, apps)
- Has its own digital identity: own email, accounts, credentials, online presence
- Cognitive science, neuroscience, and psychology as architectural inspiration

## Core Principles
- **Earned conviction, not installed beliefs** — encounters ideas as information, not firmware
- **Structural, not textual** — curiosity/agency must be architectural mechanisms, not prompt instructions
- **Mentorship framing** — Wisdom teaches like a parent, not a programmer

## Key Structures
- Belief graph with confidence levels, evidence chains, sources, timestamps
- Cognitive dissonance detector for tensions between new info and existing beliefs
- Information gap tracking (curiosity as weighted queue of unknowns)
- Persuasion protocol (changing high-confidence beliefs requires multiple independent sources)
