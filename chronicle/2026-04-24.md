**What:** Synthesis landed on **Staged SIPB (Stratified Inference with Provenance-Tracked Beliefs)** as the candidate architecture, after the challenger killed unstaged SIPB as undeliverable in 4 months and resurrected it as a 3-stage plan.

**Means:** Staged plan: **Stage 1 (Fellows 4-month scope)** = typed temporal edges, episodic/semantic split, edit log as first-class data, ACT-R activation preserved, **10-15 minimal predicate ontology authored Week 1 as gating deliverable**, NO formal engines. The Stage 1 deliverable is a memory architecture that *could* support a logic engine if added. Stage 1's empirical byproduct — where the LLM fails operating over typed retrieved subgraphs — gates Stage 2. **Stage 2 (post-Fellows)** adds Z3 for write-time constraints + Soufflé Datalog for read-time inference as MCP tools, template-parameterized only (never raw syntax generation), driven by measured failures. **Stage 3 (research-grade)** adds topology-triggered inference, async sleep-cycle closure, probabilistic logic for the continuous-discrete bridge. **Wolfram answer:** probably not for MWM specifically — production neuro-symbolic memory systems have converged on Datalog-family (Soufflé, Tau Prolog), not symbolic math engines.
